# 发票管家 — 使用说明

## 启动（两种方式任选）

### 方式一：双击 app.bat → 独立应用窗口
无浏览器边框，像真正的桌面软件一样

### 方式二：双击 发票管家.exe + 浏览器访问
启动后打开浏览器，访问 http://localhost:3456

## 默认账号

管理员：admin / admin123
新用户点"注册"创建独立账号

## 无需安装任何东西

- 不需要 Node.js
- 不需要 npm install
- 不需要数据库
- 不需要联网
- Windows 10/11 自带 Edge 即可（app.bat 需要 Edge）

## 注意事项

- exe 放在哪个文件夹，数据就存在哪个文件夹的 db/ 目录
- 不要放在 C:\Program Files（需要管理员权限）
- 建议放桌面或 D 盘
